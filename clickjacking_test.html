Prueba de Vulnerabilidad: Clickjacking (Inyección por iframe)

Este archivo simula un sitio web atacante malicioso que intenta incrustar la aplicación Helarate dentro de un iframe para realizar un ataque de Clickjacking.

Pasos para obtener la evidencia:

  1. Abre este archivo HTML en tu navegador Google Chrome.
  2. Presiona la tecla F12 (o haz clic derecho y selecciona Inspeccionar) y ve a la pestaña Console (Consola).
  3. Comprueba que el iframe de abajo se muestre completamente en gris/blanco y que no cargue la aplicación.
  4. En la consola de F12 deberás ver un mensaje de error en color rojo de tipo:
    Refused to display 'https://helarate-origin.vercel.app' in a frame because it set 'Content-Security-Policy' to 'frame-ancestors 'self''.
  5. Toma una captura de pantalla completa de tu navegador que muestre el iframe bloqueado y el error en la consola de Chrome.
